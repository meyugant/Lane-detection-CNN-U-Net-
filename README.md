# Real-Time Lane Detection using CNN

## Overview
A Computer Vision project that uses a Convolutional Neural Network (CNN) built in TensorFlow to detect road lanes from images and video streams.

## Features
- Lane segmentation
- Synthetic-to-real training
- TensorFlow implementation
- Image preprocessing
- Real-world testing

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy
- Matplotlib

## Workflow
Synthetic Dataset
↓
CNN Training
↓
Fine Tuning on Real Road Data
↓
Lane Prediction

## Results
Achieved robust lane detection on real-world road images.
